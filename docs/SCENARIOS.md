# 8 главных сценариев работы модуля White-sheet-breaker

> Документ содержит 8 ключевых сценариев для демонстрации работы модуля.

---

## Настройки для демонстрации

Для удобства демонстрации в коде изменены пороги фонового отслеживания активности:

| Параметр | Значение по умолчанию | Значение для демонстрации |
|---|---|---|
| **Уровень помощи 1** | 60 секунд (1 минута) | 30 секунд |
| **Уровень помощи 2** | 180 секунд (3 минуты) | 60 секунд |
| **Таймаут сессии** | 300 секунд (5 минут) | 90 секунд |
| **Порог активности** | 100 символов | 25 символов |

Это позволяет быстро показать все ключевые возможности модуля без длительного ожидания.

## Сценарий 1: Полный цикл с таймером

**Цель:** показать параллельную работу таймера и наблюдения

**Запуск:**

```bash
python -m breaker
```

**Шаги:**

1. `[1] Создать правило с нуля`
2. Сигнал: `файл пуст`
3. Действие: `напишу класс`
4. Режим: `[1] Создать файл`
5. Файл: `classes.py`
6. Подтвердить: `y`
7. Таймер: `y`, длительность: `1` минута
8. Ничего не делать 30 сек → меню уровня 1
9. Выбрать `[2] Шаблон класса`
10. Дождаться "Работа началась"

**Ключевые моменты:**

- Параллельная работа таймера и наблюдения
- Таймер не останавливается при показе меню
- Автозавершение при активности >25 символов

## Сценарий 4: Открытие существующего файла

```bash

```

**Запуск:**

```bash

```

**Шаги:**

1. `[1] Создать правило с нуля`
2. Сигнал: `хочу отредактировать`
3. Действие: `открою файл`
4. Режим: `[2] Открыть файл`
5. Файл: `existing.py`
6. Подтвердить: `y`

**Ключевые моменты:**

- Файл открывается в редакторе
- Тип действия: open_file
- Кроссплатформенность (macOS → `open`, Windows → `startfile`)

## Сценарий 6: Редактор шаблонов

**Цель:** показать CRUD шаблонов

**Запуск:**

```bash
python -m breaker
```

**Шаги:**

1. `[3] Управление шаблонами`
2. `[2] Создать шаблон`
3. Название: `Мой шаблон`
4. Сигнал: `нужен тест`
5. Действие: `создам pytest`
6. Цель: `test_*.py`
7. Тип: `[2] create_test`

- Сохранение в `~/.white-sheet-breaker/templates.json`

## Сценарий 7: Все 3 уровня помощи

**Цель:** показать конечный автомат наблюдения

**Запуск:**

```bash
python -m breaker
```

**Шаги:**

1. Создать правило для `test_example.py`
2. Таймер: `n`
3. Ждать 30 сек → меню уровня 1
4. Выбрать `[5] Ничего, продолжу сам`
5. Ждать 60 сек → меню уровня 2
6. Выбрать `[6] Ничего, продолжу сам`
7. Ждать 90 сек → таймаут

**Ключевые моменты:**

- Уровень 1 (30 сек): мягкая помощь — шаблоны, TODO, таймер
- Уровень 2 (60 сек): микрошаги — TODO, pass, assert, открыть README
- Таймаут (90 сек): завершение сессии
- Конечный автомат: IDLE → HELP_1 → HELP_2 → TIMEOUT
