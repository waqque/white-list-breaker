## 3. Конфигурация

### Переменные окружения

| Переменная | Значение по умолчанию | Описание |
|---|---|---|
| `LRS_URL` | `http://localhost:8080/xAPI` | URL хранилища записей |
| `LRS_KEY` | — | Ключ аутентификации LRS |
| `LRS_SECRET` | — | Секрет аутентификации LRS |
| `LRS_MODE` | `mock` | Режим работы: `mock` или `real` |
| `LEARNER_ID` | `learner_unknown` | Идентификатор учащегося |
| `LEARNER_NAME` | `Learner` | Имя учащегося |
| `COURSE_ID` | — | Идентификатор курса |

### Режимы работы LRS

| Режим | Поведение |
|---|---|
| `mock` | xAPI-стейтмент выводится в stdout как JSON |
| `real` | Стейтмент отправляется по HTTP POST в реальный LRS |

## 5. Кроссплатформенность

| Функция | Windows | macOS | Linux | WSL |
|---|---|---|---|---|
| Открытие файла | `os.startfile()` | `open` | `xdg-open` | `wslpath` + `powershell.exe` |
| Воспроизведение звука | `powershell` + `SoundPlayer` | `afplay` | `paplay`/`mpv`/`ffplay`/`aplay` | `powershell.exe` + WSL-path |
| Кодировка файлов | `encoding='utf-8'` явно | `encoding='utf-8'` | `encoding='utf-8'` | `encoding='utf-8'` |
| Пути | Нормализация `\` → `/` | `Path` | `Path` | `wslpath` для конвертации |
